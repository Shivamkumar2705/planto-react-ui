A modern, responsive, and aesthetic User Interface for a plant e-commerce website built with **React** and **Tailwind CSS**. This project features a glassmorphism design style, smooth animations, and a fully responsive layout.

## üöÄ Features

-   **Modern UI/UX:** Clean, dark-themed aesthetic with a focus on "Glassmorphism".
-   **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile screens.
-   **Dynamic Carousel:** Custom-built image slider for the "Best O2" section with floating 3D effects.
-   **Interactive Components:**
    -   Hover effects on product cards.
    -   Mobile Navigation Menu with smooth transitions.
    -   Testimonial/Review cards with unique asymmetrical border styling.
-   **Tech Stack:**
    -   [React.js](https://reactjs.org/) (Vite)
    -   [Tailwind CSS](https://tailwindcss.com/)
    -   [React Icons](https://react-icons.github.io/react-icons/)
